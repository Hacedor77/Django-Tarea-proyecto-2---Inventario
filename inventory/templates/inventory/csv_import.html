Instrucciones para el archivo CSV

El archivo CSV debe contener las siguientes columnas obligatorias:

  • code: Código único del producto
  • name: Nombre del producto
  • category: Nombre de la categoría
  • supplier: Nombre del proveedor
  • unit_price: Precio unitario (formato: 12.50)

Columnas opcionales:

  • description: Descripción del producto
  • minimum_stock: Stock mínimo (por defecto: 10)
  • maximum_stock: Stock máximo (por defecto: 1000)
  • current_stock: Stock actual (por defecto: 0)
Ejemplo de archivo CSV
code,name,description,category,supplier,unit_price,minimum_stock,maximum_stock,current_stock

PROD003,Teclado mecánico,Teclado gaming,Accesorios,Razer,89.99,10,100,45
